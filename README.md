# Automatizacion en atajos IOS

- Automatizacion en atajos IOS para movimientos financieros con ayuda del registro de Google Sheets, Notion Y Numbers.
- En este manual aunque los procedimientos sean similares depende de la plataforma que desees usar se necesitarán diferentes cosas.
- Es necesario contar con un Iphone y el app de Atajos para estas automatizaciones.
- En este manual no se brindará una guía a profundidad del dashboard ni de como organizar tus categorías, solo se explicará la intengración con la base de datos en cualquier de las 3 plataformas memcionadas.
